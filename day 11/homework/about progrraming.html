ეს გვერდი შეიცავს პროგრამირების შესახებ ინფორამაციას.

  • როგორ ვისწავლოთ პროგრამირება მარტივად?

    • პროგრამირების პირველი ენაა python.პითონი არის მაღალი დონის, ზოგადი დანიშნულების პროგრამირების ენა. მისი დიზაინის ფილოსოფია ხაზს უსვამს კოდის წაკითხვას მნიშვნელოვანი ჩაღრმავების გამოყენებით. პითონი არის დინამიურად აკრეფილი და ნაგვის შეგროვება. იგი მხარს უჭერს პროგრამირების მრავალ პარადიგმას, მათ შორის სტრუქტურირებულ (განსაკუთრებით პროცედურულ), ობიექტზე ორიენტირებულ და ფუნქციურ პროგრამირებას. მას ხშირად აღწერენ, როგორც "ბატარეებს მოყვება" ენას მისი ყოვლისმომცველი სტანდარტული ბიბლიოთეკის გამო. გვიდო ვან როსუმმა დაიწყო პითონზე მუშაობა 1980-იანი წლების ბოლოს, როგორც ABC პროგრამირების ენის მემკვიდრე და პირველად გამოუშვა იგი 1991 წელს, როგორც Python 0.9.0.[36] Python 2.0 გამოვიდა 2000 წელს. Python 3.0, გამოშვებული 2008 წელს, იყო ძირითადი რევიზია, რომელიც სრულიად არ იყო თავსებადი წინა ვერსიებთან. Python 2.7.18, გამოშვებული 2020 წელს, იყო Python 2-ის ბოლო გამოშვება. პითონი მუდმივად იკავებს პროგრამირების ერთ-ერთ ყველაზე პოპულარულ ენას და ფართო გამოყენებას მოიპოვებს მანქანათმცოდნეობის საზოგადოებაში.
